## Overview

Windows medium box. Attack chain: ASREPRoasting to get a hash, crack it, use credentials to explore XMPP (Jabber/Openfire), find more credentials in chat history, DCOM lateral movement, Openfire plugin privesc to SYSTEM.

## Foothold

### ASREPRoasting

No pre-auth required on one account. Grabbed the hash, cracked offline with hashcat.

```bash
impacket-GetNPUsers domain.htb/ -usersfile users.txt -no-pass -dc-ip 10.10.11.x
hashcat -m 18200 hash.txt /usr/share/wordlists/rockyou.txt
```

### XMPP enumeration

Credentials worked on the Openfire/Jabber service. Used Profanity to connect and browse message history. Found plaintext credentials for a higher-privileged account in an old conversation.

## Lateral Movement

DCOM execution via `MMC20.Application` using the recovered credentials.

```bash
impacket-dcomexec domain.htb/user:password@10.10.11.x 'cmd /c whoami'
```

## Privilege Escalation

Openfire running as SYSTEM. Uploaded a malicious plugin via the admin console to get command execution. Dropped a shell.

## Notes

- dnSpy useful for inspecting the Openfire plugin format
- Profanity is the cleanest CLI XMPP client for this kind of enumeration
- The DCOM path was obvious once you had the second set of creds — SYSTEM on the target machine had Openfire admin access
